## Media Query

Mediaquery effects for different screens mobile, tablet and laptop.


/* Tablet CSS rules */
@media only screen and (min-width: 426px) {
    body {
      background-color: blue;
    }
  }

/* Desktop CSS rules */
@media only screen and (min-width: 769px) {
    body {
      background-color: black;
    }
  }
